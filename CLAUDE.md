# GG Portfolio Site

## Stack
- Next.js 16 (App Router, static export)
- React 19, Tailwind v4, Motion (Framer Motion)
- TypeScript strict mode

## Structure
- `src/app/` - Pages (/, /about, /projects, /writings)
- `src/components/` - Shared components
- Static export configured in `next.config.ts`

## Deployment
- GitHub: `Machoman3000/gg-portfolio`
- Vercel: `vercel --prod` to deploy
- Auto-deploy not configured (needs GitHub Login Connection in Vercel dashboard)

## Design System
- Cyberpunk monochrome: `#050505` bg, `#00ff41` accent
- Fonts: JetBrains Mono (code), Space Grotesk (headings)
- Corner brackets, left accent bars on cards
